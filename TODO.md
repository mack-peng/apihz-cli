## 快捷添加流程

每个 API 需要：
1. 在 `src/api/<name>.ts` 中新增 API 类方法（参考现有代码模式）
2. 在 `src/commands/<name>.ts` 中注册命令
3. 在 `src/commands/index.ts` 中导出
4. 在 `src/program.ts` 中注册
5. `npm run build` 编译
